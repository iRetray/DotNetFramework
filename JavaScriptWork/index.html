TALLER DE DESARROLLO DE CONCEPTOS DE LENGUAJE DE PROGRAMACIÓN JAVASCRIPT PARA LA TRANSFORMACIÓN DE DOCUMENTOS WEB CON HTML Y CSS3

Julian Camilo Cruz Sanchez

1. Desarrollar un documento Web que permita cambiar el estilo de un párrafo de texto, pasando el puntero sobre el párrafo, cambiando su color y tamaño de fuente 2. Desarrollar un documento Web que permita ingresar por un prompt prompt ("mensaje") un número entero e imprimir sobre un span la impresión separada con espacios en blanco de los números del 1 al número ingresado 3. Elaborar un documento Web que reciba por pantalla a través de un prompt, una cadena de caracteres y usando la propiedad innerHTML despliegue el texto ingresado en los seis tipos de título existentes para documentos de la Web. Cada título debe llevar un color CSS diferente, empleando la propiedad style de la etiqueta para modificar el color de cada título. 4. Elaborar un formulario Web HTML que reciba los valores de los lados de un triángulo rectángulo y mediante la fórmula del teorema de Pitágoras establezca que los lados de los catetos opuestos y adyacente, coinciden por fórmula, con el valor de la hipotenusa que haya sido ingresada.
